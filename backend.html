`backend` — DHT and key management¶

This package contains classes for connecting to the DHT and managing keys and certificates.

Modules:

  • certManager — Manages certificates
  • dhtServer — Wrapper around key value storage
  • keyManager — Manages keys
  • localServer — Control the program
  • server — Main entry point to start the server
  • storage — List storage
  • verifier — Find names for keys
