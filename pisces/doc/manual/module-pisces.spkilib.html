5.11 pisces.spkilib - SPKI package

The SPKI implementation is a contained in the pisces.spkilib sub-package. The core of the implementation, contained in pisces.spkilib.spki, defines Python objects for each of the SPKI/SDSI 2.0 objects defined in the last draft of the structure document [1].
