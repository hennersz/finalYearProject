Installation
============

Prerequisites
-------------
* Python 2.7
* Pip
* Git

Install
-------

::

    $ git clone https://github.com/hennersz/finalYearProject.git
    $ cd finalYearProject
    $ pip install -r requirements.txt
    $ cp example.cfg ~/.p2ppki/config.cfg


Test
---- 
Run tests using tox::

  $ tox
